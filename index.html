Abstract

‡∏ö‡∏ó‡∏Ñ‡∏±‡∏î‡∏¢‡πà‡∏≠ (‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç/‡∏¢‡πà‡∏≠/‡πÅ‡∏¢‡∏Å‡∏¢‡πà‡∏≠‡∏´‡∏ô‡πâ‡∏≤‡πÑ‡∏î‡πâ)

Over 41 million people worldwide die from NCDs (non-communicable diseases) annually, the majority of which occur in low- and middle-income countries. Social media has become a critical platform for individuals to share experiences and access information about NCDs. Social (media) listening offers valuable insights by analyzing user discussions, but existing tools are closed-source and commercial.

This study seeks to simplify the extraction of NCD-related knowledge from social media, making it easier for the public to understand and access information. We proposed an open-source social (media) listening tool called NCDs Listener to collect, analyze, summarize, and visualize data.

Use keyword matching and the BERT Model to extract knowledge from comments. The preliminary data was analyzed using descriptive statistics. Additionally, a Generative AI model summarizes the extracted knowledge in human-readable sentences.

Our NCDs Listener tool is open-source and can extract knowledge related to NCDs. This knowledge can be used as a guideline for treatment or the development of effective care to meet patients‚Äô needs.

Workflow

5-step pipeline: scraping ‚Üí preprocessing ‚Üí extraction ‚Üí adjustment ‚Üí visualization

1

Data Scraping

User enters a post URL ‚Üí Selenium + BeautifulSoup extract comments and metadata from public posts.

2

Data Preprocessing

Remove duplicates, filter extremely short text, normalize formatting, export CSV for downstream analysis.

3

Knowledge Extraction

Tokenization, stopwords, lemmatization + keyword matching + BERT classification to identify comment types.

4

Data Adjustment

Users include/exclude diseases and symptoms ‚Üí system filters the extracted knowledge accordingly.

5

Data Visualization

Interactive dashboard + GenAI summarization; summary stats table; export results as PDF.

Systems Design

Implementation overview (Python 3.11.9 + NLP + dashboard)

Architecture

  • Input: Social post URL + include/exclude rules
  • Scraping: Selenium/BS4 (public content)
  • NLP: Keyword matching + BERT classification
  • Summarization: Generative AI produces human-readable insights
  • Output: Dashboard + CSV/PDF exports
